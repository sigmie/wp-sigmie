== Description ==

[Sigmie](https://sigmie.app) is a search tool that improves your users' search experience by providing quick and accurate results, before they finish typing.

It offers features like synonyms, typo tolerance, and highlighting and with it's easy integration, it solves any search issues in WooCommerce.

A [free trial](https://app.sigmie.com/sign-up) is available (no credit card needed).
